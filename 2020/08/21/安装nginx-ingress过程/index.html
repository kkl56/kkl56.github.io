安裝nginx-ingress過程

安装nginx-ingress过程

官方文档

3个步骤：

1.下载镜像 以及deployment文件

docker pull nginx/nginx-ingress:1.8.1
git  clone https://codeload.github.com/nginxinc/kubernetes-ingress/zip/release-1.8
cd  kubernetes-ingress/deployments

2.部署deployment

kubectl apply -f common/ns-and-sa.yamlkubectl apply -f rbac/rbac.yaml
kubectl apply -f rbac/rbac.yaml
kubectl apply -f common/default-server-secret.yaml
kubectl apply -f common/nginx-config.yaml

#如果报错，apply时加参数 --validate=false

kubectl apply -f common/vs-definition.yaml
kubectl apply -f common/vsr-definition.yaml
kubectl apply -f common/ts-definition.yaml
kubectl apply -f common/policy-definition.yaml

kubectl apply -f common/gc-definition.yaml
kubectl apply -f common/global-configuration.yaml

#这里使用daemon-set方式创建，注意里面的image版本要与准备好的的image对应，否则会去网上pull。
kubectl apply -f daemon-set/nginx-ingress.yaml

3.检查

kubectl get pods --namespace=nginx-ingress
